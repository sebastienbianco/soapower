Upload Configuration

Upload a CSV file containing Environments and Services data. Be aware of the following rules :

  • You can download a file containing current data with "Download configuration" button
  • Uploading a file only creates or updates existing data
  • An environment and a service are identified by their ids
  • If the id is empty, the environment and the service will be either created or updated depending on the other data
  • If the id is not empty and exist in db, it will be updated. If it does not, it will be created
  • Uploading stops and fail at first error occurred. Fix the error(s) and upload again
